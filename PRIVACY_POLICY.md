# Privacy Policy for Speed Test Chrome Extension

**Last Updated:** [Date]

## 1. Information We Collect

This extension collects and processes the following information:

### Network Performance Data:
- Download and upload speeds
- Latency measurements
- Connection type (WiFi, cellular, etc.)

### Basic Network Information:
- IP address (used only for speed testing, not stored)
- DNS information
- Approximate location (based on IP, not precise)

## 2. How We Use Information

All data is used solely for:
- Providing speed test results to you
- Displaying network information in the extension
- Improving the accuracy of speed measurements

## 3. Data Storage

- All test results are stored locally in your browser
- No personal data is transmitted to our servers
- Network information is cached temporarily for performance

## 4. Data Sharing

We do not:
- Share your data with third parties
- Sell or monetize your information
- Track your browsing activity

## 5. Security

We implement:
- HTTPS for all external requests
- Minimal data collection
- Local storage only (no server databases)

## 6. Third-Party Services

The extension uses:
- Cloudflare's speed test API (their privacy policy applies)
- Chrome's storage.sync API for settings

## 7. Changes to This Policy

We may update this policy - you'll be notified within the extension.
